# UI Specifications

## Localization

### Right-to-Left (RTL) Support

The application must support RTL (Right-to-Left) layout because it's designed for Hebrew users. All UI components, text alignment, and navigation elements should be properly mirrored for RTL languages.

## Design System

### Brand Colors

- Primary Color: Ocean Blue (#0077be)

### Typography

- Primary Font: Assistant

The Assistant font family should be used throughout the application for optimal readability in Hebrew text.
